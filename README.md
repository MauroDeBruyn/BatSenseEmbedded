# BatSenseEmbedded

Embedded sensor platform for monitoring bat activity inside cavity walls.  
Built around an ESP32, Raspberry Pi Pico, IR cameras, and audio modules to support ecological studies and renovation planning.

## Focus
- Integration with Home Assistant  
- Direct data retrieval from modules using an ESP32 with Wi-Fi and SD storage
